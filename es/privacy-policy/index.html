Política de Privacidad

Gracias por leer esta página: significa que te importa tu privacidad.

Resumen

Esta Política de Privacidad describe cómo https://demdxx.com (el Sitio Web) trata la información.

Qué información tratamos

Información que no envías a través de este Sitio Web

No recopilamos datos personales que introduzcas en el Sitio Web, porque no hay formularios ni procesos de registro. Si deseas contactarme, puedes usar los enlaces externos del sitio (redes sociales) o escribirme por email.

Información recopilada automáticamente

Al visitar el Sitio Web, los servicios de alojamiento y analítica pueden procesar datos técnicos como:

  • Dirección IP (a menudo truncada o anonimizada)
  • Tipo y versión del navegador
  • Sistema operativo
  • URL de referencia
  • Páginas visitadas y tiempo aproximado de visita
  • Fecha y hora de acceso

Esta información ayuda a entregar el Sitio Web, mantenerlo seguro y comprender el uso agregado.

Google Analytics

Utilizamos Google Analytics para medir estadísticas de tráfico anónimas o seudonimizadas. Google puede usar cookies y procesar datos según sus propios términos. Más información en la Política de Privacidad de Google.

Puedes limitar el seguimiento analítico mediante la configuración del navegador, herramientas de bloqueo o el complemento de inhabilitación de Google Analytics.

Para más detalles sobre cookies, consulta nuestra Política de cookies.

Enlaces externos

El Sitio Web enlaza a plataformas de terceros como LinkedIn, GitHub, Instagram y Facebook. Si sigues esos enlaces, esos servicios tratan tus datos según sus propias políticas de privacidad. No controlamos ni somos responsables de sitios web de terceros.

Base legal (RGPD)

Cuando se aplica el Reglamento General de Protección de Datos (RGPD), el tratamiento se basa en:

  • Interés legítimo — operar, proteger y mejorar un sitio personal informativo mediante estadísticas agregadas
  • Consentimiento — cuando sea necesario para cookies no esenciales u otras tecnologías similares (puedes rechazarlo o retirarlo en la configuración del navegador)

Conservación de datos

Los datos analíticos se conservan según la configuración de Google Analytics. Los registros del servidor o CDN, si los hay, se conservan solo el tiempo necesario para seguridad y funcionamiento.

Tus derechos

Si la ley aplicable te otorga derechos sobre tus datos personales (por ejemplo, bajo el RGPD), puedes solicitar acceso, rectificación, supresión, limitación, oposición o portabilidad respecto a los datos que controlamos.

Para ejercer tus derechos, contacta con demdxx@gmail.com. También puedes presentar una reclamación ante tu autoridad de protección de datos.

Transferencias internacionales

Proveedores como GitHub (alojamiento) y Google (analítica) pueden procesar datos en países distintos al tuyo. El tratamiento está sujeto a sus términos y a las garantías legales aplicables.

Seguridad

Aplicamos medidas razonables apropiadas para un sitio web estático. No obstante, ninguna transmisión por internet es completamente segura.

Menores

Este Sitio Web no está dirigido a menores de 16 años y no recopilamos conscientemente sus datos personales.

Cambios

Podemos actualizar esta Política de Privacidad cuando cambie el Sitio Web o la legislación aplicable. La versión vigente siempre se publica en esta página.
